Exportar Llibre IGI
===================

Amb aquest mòdul podem exportar a Excel els llibres d'IGI corresponents de factures rebudes i emeses. S'ha de tenir en compte que totes les línies de factura han de tenir mínim / màxim un impost.

L'exportador tindrà en compte les factures obertes i les pagades segons el filtre assignat.

Instal·lació
------------

Aquest mòdul depen del paquet openpyxl. Cal instal·lar el fitxer de requeriments.
